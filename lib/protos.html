Union-find (weighted + path compression)

A forest in one array: parent[i]. Union by size keeps trees log-shallow; compression flattens them to ~constant.

int find(int x) {
    int root = x;
    while (parent[root] != root) root = parent[root];
    // TODO: path compression — repoint every node on the walked path to root
    return root;

void unite(int p, int q) {
    int rp = find(p), rq = find(q);
    if (rp == rq) return;
    if (size[rp] > size[rq]) swap(rp, rq);   // rp = smaller
    parent[rp] = rq;
    size[rq] += size[rp];
}

Substring search — KMP

Never back up in the text: on a mismatch, the failure function says how much of the pattern still matches itself.

int kmpSearch(text, pat) {
    f = failure(pat);
    int j = 0;                                     // chars of pat matched
    for (int i = 0; i < text.length(); i++) {
        while (j > 0 && text[i] != pat[j]) j = f[j-1];   // slide the pattern
        if (text[i] == pat[j]) j++;
        if (j == pat.length()) return i - j + 1;   // match ends at i
    }
    return -1;
}
