At a glance

You implement the two divide-and-conquer sorts from the reading — merge/mergesort and partition/quicksort — plus quickselect, the partition-based Θ(n) selection algorithm. The starter gives the data structures, helpers, and a self-running test battery; you fill the TODOs.

Tasks

  1. merge — combine two adjacent sorted runs using the auxiliary buffer.
  2. mergesort — recurse on halves, then merge.
  3. partition — place a pivot so nothing larger sits left of it, nothing smaller right; return its index.
  4. quicksort — partition, then recurse on both sides.
  5. quickselect — recurse only into the side that contains index k.

The tests run themselves

g++ -std=c++17 -g ica13.cpp -o ica13 && ./ica13
  • T1–T2 mergesort — known array + random batches vs std::sort
  • T3–T4 quicksort (random, duplicates, sorted) + the partition invariant
  • T5 quickselect(k) == sorted[k] for several k
  • T6 edge cases — single element, all-equal
